Hệ thống Hadoop Cluster

1. Kiến trúc Cluster

Sử dụng Docker để giả lập cụm 5 máy chủ (Node):

  • Namenode: Máy chủ quản lý.
  • Datanode: Máy chủ lưu trữ dữ liệu.
  • ResourceManager: Quản lý tài nguyên tính toán.
