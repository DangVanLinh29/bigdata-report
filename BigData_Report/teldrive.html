Triển khai Hệ thống TeleDrive

TeleDrive là giải pháp biến Telegram thành ổ đĩa đám mây không giới hạn. Dưới đây là hướng dẫn cài đặt chi tiết.

1. Cấu hình Docker Compose

Tạo file docker-compose.yml với nội dung sau (đã ẩn các mã bí mật):


version: '3.9'
services:
  teldrive:
    image: ghcr.io/tgdrive/teldrive:latest
    container_name: teldrive-core
    restart: always
    ports:
      - "8080:8080"
    command: >
      run
      --tg-app-id=YOUR_APP_ID
      --tg-app-hash=YOUR_APP_HASH
      --jwt-secret="secret_key"

2. Kết quả cài đặt

Sau khi chạy lệnh, truy cập localhost:8080 để đăng nhập.
