CÀI ĐẶT TELEDRIVE

Quay lại Trang chủ

1 Chuẩn bị Thư mục

Tạo cấu trúc thư mục dự án trên máy tính:

/PTDLL
├── teldrive_data/      (Tự động tạo khi chạy)
├── pg_data/            (Tự động tạo khi chạy)
└── docker-compose.yml  (File cấu hình chính)

2 Lấy API Credentials từ Telegram

Để TeleDrive kết nối được với tài khoản Telegram, bạn cần đăng ký ứng dụng:

  1. Truy cập: https://my.telegram.org
  2. Nhập số điện thoại của bạn và mã xác nhận được gửi về Telegram.
  3. Chọn mục API development tools.
  4. Điền thông tin vào form:
    • App title: Đặt tên bất kỳ (ví dụ: MyBigDataDrive).
    • Short name: Đặt tên ngắn (ví dụ: mybigdatadrive).
    • Platform: Chọn Desktop.
  5. Bấm Create application.
  6. Copy 2 dòng quan trọng: App api_id và App api_hash.
Lưu ý: Giữ bí mật 2 mã này, không chia sẻ cho người lạ.

3 Cấu hình Docker Compose

Tạo file docker-compose.yml và điền API ID/Hash bạn vừa lấy được vào:

version: '3.9'

services:
  db:
    image: groonga/pgroonga:latest-alpine-16
    container_name: teldrive-db
    restart: always
    environment:
      POSTGRES_USER: teldrive
      POSTGRES_PASSWORD: teldrive_password
      POSTGRES_DB: teldrive
    volumes:
      - ./pg_data:/var/lib/postgresql/data
    networks:
      - teldrive-net

  teldrive:
    image: ghcr.io/tgdrive/teldrive:latest
    container_name: teldrive-core
    restart: always
    depends_on:
      - db
    ports:
      - "8080:8080"
    # Thay thế YOUR_APP_ID và YOUR_APP_HASH bằng số bạn vừa lấy ở Bước 2
    command: >
      run
      --tg-app-id=YOUR_APP_ID
      --tg-app-hash=YOUR_APP_HASH
      --db-data-source="postgres://teldrive:teldrive_password@db:5432/teldrive?sslmode=disable"
      --jwt-secret="random_secret_string"
      --tg-uploads-encryption-key="chuoi_32_ky_tu_bat_ky_do_ban_chon"
    volumes:
      - ./teldrive_data:/root/.teldrive
    networks:
      - teldrive-net

networks:
  teldrive-net:
    driver: bridge

4 Khởi chạy & Kết quả

Chạy lệnh sau trong Terminal:

docker-compose up -d

Sau đó truy cập http://localhost:8080. Đăng nhập và vào Settings -> Channels để chọn kênh lưu trữ mặc định.
