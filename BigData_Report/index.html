Giải pháp kết hợp sức mạnh lưu trữ vô hạn của TeleDrive và khả năng tính toán phân tán của Hadoop Cluster.

TeleDrive

Biến Telegram thành ổ đĩa mạng không giới hạn dung lượng. Hỗ trợ upload file lớn, mã hóa dữ liệu và quản lý qua giao diện Web.

Hadoop Cluster

Hệ thống xử lý phân tán với kiến trúc 5 node (Namenode, Datanode, YARN...). Chạy trên nền tảng Docker container hóa.

Kiến trúc Hệ thống

Containerization

Toàn bộ hệ thống được đóng gói bằng Docker, đảm bảo tính nhất quán và dễ dàng triển khai trên mọi môi trường (Windows/Linux).

Bảo mật dữ liệu

Dữ liệu upload lên Telegram được mã hóa đầu cuối thông qua TeleDrive, đảm bảo an toàn tuyệt đối cho Dataset.

Phân tích MapReduce

Sử dụng Hadoop Streaming để chạy các thuật toán phân tích dữ liệu viết bằng Python/Java.
